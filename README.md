# SweetCrateKeys

> /keyupgrade [-p Player] - Opens the Key Upgrade Menu - **sweetcratekeys.use**

__upgrades.yml__
```yml
# Only works with ExcellentCrates
upgrades:
  upgrade_1:
    name: Vote Upgrade
    tradeInKeyID: vote
    tradeInKeyAmount: 3
    rewards:
      chocolate:
        chance: 15
      taffy:
        chance: 35
      peppermint:
        chance: 50
  upgrade_2:
    name: Rare Upgrade
    tradeInKeyID: rare
    tradeInKeyAmount: 3
    rewards:
      legendary:
        chance: 15
      seasonal:
        chance: 85
```
